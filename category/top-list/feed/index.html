Top List – Fullsnack Developer https://fullsnackdeveloper.com Just another WordPress site Tue, 31 Mar 2020 15:20:29 +0000 en-CA hourly 1 https://wordpress.org/?v=5.3.4 Top 5 Udemy Courses for Coding of 2019 https://fullsnackdeveloper.com/2019/07/15/top-5-udemy-course-suggestions-for-coding-of-2019/ https://fullsnackdeveloper.com/2019/07/15/top-5-udemy-course-suggestions-for-coding-of-2019/#respond Mon, 15 Jul 2019 16:01:28 +0000 https://fullsnackdeveloper.com/?p=1682 You want to learn to program but don’t know where to start. You want to level up your skills but you don’t have a mentor. You want to learn how to make an application with authentication. The list can go on and on and they all have the same answer; tutorials.

There are piles of free and paid tutorials out there which makes it hard to find the really good ones. I have gone through countless tutorials on multiple different sites but the one I keep going back to is Udemy.

Udemy has taught me so much and I attribute a huge amount of my success to all the awesome courses that they offer. You’ll notice that I prefer certain instructors and you can find them at the end of this article. I would recommend any of the courses they offer.

Advanced JavaScript

There are a lot of free hello world tutorials out there for learning Javascript but after learning the basics I was looking for a way to level up. This course takes you through and explains the basics as well as advanced concepts in JavaScript. Understanding even the basics in a detailed way helped me understand how the language works on a deeper level. If you’re looking to level up your JS skills this course if for you.

Node with React

You guys know how much I love React. Well, this course was the beginning for me. I learned from start to finish, project structure, life cycle methods, processing payments, setting up a node server and all the rest. This is an incredibly thorough course that breaks down even the most complex concepts into understandable bites. If you want to learn React this is the one I would recommend the most. As a note, he uses a Mongo DB for the database which I recommend as well and I have a course for it further down.

React Redux Firebase

Working with Mongo is awesome but it does require a backend server so when I found out about Firebase and that its serverless I hoped on. And obviously, I’m going to use React as the frontend but state management can be a lot but Redux is so good for it. This course shows how to make them all work together and the flow has been one that I’ve used in multiple projects. If you’re looking to work with these frameworks, I would highly recommend this course.

Mongo DB

One of the first web development jobs was working on an application that used Mongo DB it holds a special place in my heart. And I have used other databases but I always come back to Mongo. This course shows you how to manage and utilize all the amazing capabilities that Mongo DB offers. He does a great job of explaining what he’s doing as he’s doing it and is really easy to follow. If you’re going to start using Mongo DB, this course is what you need.

Angular 8

Now you guys know I’m all about React but recently I had to learn Angular 8 for a project. Although I was reluctant, tools and tools and I’ve learned not to smack talk other frameworks and actually gained an appreciation for Angular. This was the course that I crashed through to ramp on the project and it was a great experience. This course takes you from start to finish on how to use the framework as well as all the tools that the CLI offers to make development that much faster. So If you want to get started in Angular this course if a home run.

Bonus Ionic 4

For the project that I was referring to in the above article, we were also using ionic to develop. So Ionic 4 was on the roster for the ramp up and obviously, I went straight to Udemy. This course was an awesome find. After going through it myself I sent it to my coworkers and they also found it super helpful. He shows you all the capabilities of the framework and how to get your project showing up on your phone in no time. If you’re unfamiliar with Ionic, it’s a framework that turns your web development project into a native app. It originally was only for Angular but recently just released support for React and Vue. So if you want to turn to web development skills into native app development this course is a great option.

Recommended Instructors

My two favorite instructors on Udemy are Stephen Grider and Maximilian Schwarzm√ºller. Both do an incredible job of explaining complex concepts in an understandable way, create interesting and useful projects that incorporate all the lessons taught and are easy to follow. If I haven’t recommended a course that is in your field of interest, I would recommend any of the courses they offer. Both have a vast collection of videos in a plethora of different programming languages.

Conclusion

I hope you find these suggestions helpful and if I’ve missed a great course please leave a comment with a link. I would love to hear your guy’s suggestions.

“Well, the first thing I need is a code editor.”

So I googled to best code editor of the day. Through a bunch of searching I found an editor called Sublime and at that time it was. I always like to think that I started using it before it became mainstream (the hipster in me lol).

Having a design background, first things first- this needs to look prettier. And so, the search for the best theme began. To this day, although the editor itself may have changed, the desire to find a better, cooler theme still lives within me. As I’m sure it does in you as well.

Below are some of the best themes I’ve worked with. I’m a guy that likes to switch it up a lot but I’m always picking one of these themes. If I’ve missed one that you think should be on the list, please leave a comment with the name of it! I would love to check it out and maybe update this list.

Synth Wave ’84

I’m coming out the gate swinging. This is by far the coolest theme that I have ever used. Synth Wave ’84 has awesome neon colors, wild gradients and the best of all is… the text flipping GLOWS! This theme would do the new season of Stranger Things incredible justice.

The setup of this theme is a touch more involved than most but believe me it’s worth it. You can get install instructions here!

Night Owl

Night Owl was a random find when I was looking up how to make my own theme but was so good I stopped trying. This one has been my go to theme for the longest time probably because I love the idea of a theme design specifically for me, a Night Owl:)

Now I know, heaven forbid I use a light theme, but sometimes when I work outside, I have to. And the nice thing about the Night Owl theme is that it has a complimentary light theme in the same palette. Check it out here!

Material Theme

This theme dates right back to the beginning for me. My absolute favorite theme for Sublime was a material theme. As I worked my way through different editors, the material theme is always one I looked for. So naturally, when I landed on Visual Studio Code, I immediately searched for Material Theme. For Me, Material Theme is the standard with a little nostalgia so I don’t know if it can ever come off my list.

Also, you can download the accompanying icon set which takes the theme to a new level! See what you think.

Arbiter

Now, this is a particularly awesome theme because I work with the guy that made it! Now I’m pretty selective about what themes I use but Jesse (WonderPanda) nailed this one and we use it company wide! It’s easy on the eyes but still very visible. See for yourself!

Vue Theme

Obviously, if you’re a fan of Vue you’re going to love this theme. I really enjoy the color choices as well as some line highlighting to show where braces open and close. Overall its a solid theme for when I’m looking to change things up. Maybe it’ll be your fav! You can see it here.

Slack Theme (Bonus Fun)

Now I don’t know if I would seriously use this one but I thought it was hilarious when I first stumbled upon it; a Visual Studio Code theme that looks like Slack. Check it out here and let me know if you start using it full time.

Conclusion

I’ve been hunting for good themes for years and the hunt for better will always continue but hopefully, this article has been helpful and that you were able to find a Visual Studio Code theme that fit your needs and pleased your eyes. I think these are the best VS Code themes of 2019 but if you think you have a better one, through it in the comments and I’ll check it out. One of these days I’ll have to make my own:)
