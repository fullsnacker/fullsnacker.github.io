Inspiration – Fullsnack Developer https://fullsnackdeveloper.com Just another WordPress site Tue, 31 Mar 2020 15:21:38 +0000 en-CA hourly 1 https://wordpress.org/?v=5.3.4 The 5 Best Visual Studio Code Themes of 2020 https://fullsnackdeveloper.com/2019/07/11/the-5-best-visual-studio-code-themes-of-2019/ https://fullsnackdeveloper.com/2019/07/11/the-5-best-visual-studio-code-themes-of-2019/#comments Thu, 11 Jul 2019 12:13:13 +0000 https://fullsnackdeveloper.com/?p=1684 Years ago when I first wanted to learn to program, I thought,

“Well, the first thing I need is a code editor.”

So I googled to best code editor of the day. Through a bunch of searching I found an editor called Sublime and at that time it was. I always like to think that I started using it before it became mainstream (the hipster in me lol).

Having a design background, first things first- this needs to look prettier. And so, the search for the best theme began. To this day, although the editor itself may have changed, the desire to find a better, cooler theme still lives within me. As I’m sure it does in you as well.

Below are some of the best themes I’ve worked with. I’m a guy that likes to switch it up a lot but I’m always picking one of these themes. If I’ve missed one that you think should be on the list, please leave a comment with the name of it! I would love to check it out and maybe update this list.

Synth Wave ’84

I’m coming out the gate swinging. This is by far the coolest theme that I have ever used. Synth Wave ’84 has awesome neon colors, wild gradients and the best of all is… the text flipping GLOWS! This theme would do the new season of Stranger Things incredible justice.

The setup of this theme is a touch more involved than most but believe me it’s worth it. You can get install instructions here!

Night Owl

Night Owl was a random find when I was looking up how to make my own theme but was so good I stopped trying. This one has been my go to theme for the longest time probably because I love the idea of a theme design specifically for me, a Night Owl:)

Now I know, heaven forbid I use a light theme, but sometimes when I work outside, I have to. And the nice thing about the Night Owl theme is that it has a complimentary light theme in the same palette. Check it out here!

Material Theme

This theme dates right back to the beginning for me. My absolute favorite theme for Sublime was a material theme. As I worked my way through different editors, the material theme is always one I looked for. So naturally, when I landed on Visual Studio Code, I immediately searched for Material Theme. For Me, Material Theme is the standard with a little nostalgia so I don’t know if it can ever come off my list.

Also, you can download the accompanying icon set which takes the theme to a new level! See what you think.

Arbiter

Now, this is a particularly awesome theme because I work with the guy that made it! Now I’m pretty selective about what themes I use but Jesse (WonderPanda) nailed this one and we use it company wide! It’s easy on the eyes but still very visible. See for yourself!

Vue Theme

Obviously, if you’re a fan of Vue you’re going to love this theme. I really enjoy the color choices as well as some line highlighting to show where braces open and close. Overall its a solid theme for when I’m looking to change things up. Maybe it’ll be your fav! You can see it here.

Slack Theme (Bonus Fun)

Now I don’t know if I would seriously use this one but I thought it was hilarious when I first stumbled upon it; a Visual Studio Code theme that looks like Slack. Check it out here and let me know if you start using it full time.

Conclusion

I’ve been hunting for good themes for years and the hunt for better will always continue but hopefully, this article has been helpful and that you were able to find a Visual Studio Code theme that fit your needs and pleased your eyes. I think these are the best VS Code themes of 2019 but if you think you have a better one, through it in the comments and I’ll check it out. One of these days I’ll have to make my own:)

]]> https://fullsnackdeveloper.com/2019/07/11/the-5-best-visual-studio-code-themes-of-2019/feed/ 6 3 Websites to Use When Designing a Website https://fullsnackdeveloper.com/2018/12/19/3-websites-to-use-when-designing-a-website/ https://fullsnackdeveloper.com/2018/12/19/3-websites-to-use-when-designing-a-website/#comments Wed, 19 Dec 2018 16:13:18 +0000 https://fullsnackdeveloper.com/?p=1542 Finding inspiration for any design is key. Whether it’s nature, the way a building looks or from other websites. These are the 3 main websites that I use on a consistent basis when I need inspiration for a website I’m designing. 

I’ll say this right off the bat, I do not believe is ripping someone’s work off. Changing a few photos and a font isn’t enough. Finding pieces of different sites that you like and putting them together is the way to do it. 

Dribbble

Dribbble is my first choice when looking for website design inspiration. It’s a closed community of incredible designers and all the work is truly amazing. I often see cutting edge design that spurs me on to think outside the box. 

Dribbble is where designers gain inspiration, feedback, community, and jobs and is your best resource to discover and connect with designers worldwide.

Dribble Website

Dribbble is also a great place for UX inspiration. There a tons of animated gifs of small UX interactions that can inspire how users interact with your website or app.

Awwwards

This website dedicated to finding and awarding the best websites in the world. If you want to find the best in web development and design, this is the place to go. They have multiple categories to browse that always help me find inspiration in a particular area. This is a great site to find fresh thinking of websites.

Awwwards are the Website Awards that recognize and promote the talent and effort of the best developers, designers and web agencies in the world.

Awwwards Website

Designspiration

I have been using Designspiration for the longest time, ever since I starting in the design field. This site isn’t limited to digital design but all different types, architectural, photography, fashion, furniture etc. The cool thing is design inspiration can come from anywhere so the wide range of design fields can spark ideas in any area.

Designspiration is a tool for discovering great art, design, architecture, photography and web inspiration.

Designspiration Website

So these are just some of the many great websites that help inspire designers. If you have any websites that help you get inspiration, please leave the urls in the comments!
